# 숫자 야구 게임

## 진행 방법

* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정

* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 구현

* 숫자입력
    - 3자리만 받도록
    - 1~9 사이의 숫자로 구성
    - 서로 다른 숫자로 이루어지도록 확인
* 랜덤난수생성(pickNumberInRange() 활용)
    - 서로 다른 숫자로 이루어지도록 확인
    - 1~9 사이의 숫자로 구성
* 같은 숫자인지 체크
    - 같은 자리인 경우 스트라이크
    - 다른자리인 경우 볼
    - 없는경우 낫싱
* UI 구현
    - 입력 안내 메세지 출력
    - 게임 결과 출력
    - 다시시작 또는 종료
